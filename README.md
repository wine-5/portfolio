ゲーム開発の軌跡、作品、技術スキルを紹介しています。

## 🌐 デモサイト

[https://wine-5.github.io/portfolio](https://wine-5.github.io/portfolio)

## ✨ 特徴

### 🌌 サイバーパンク HUDデザイン
- **アニメーションバックグラウンド**: Canvas 2Dで描画する近未来グリッド・パーティクル・光跡
- **ダークモード対応**: システム設定に基づく自動切り替え + 手動トグル
- **View Transitions API**: テーマ切り替え時のスムーズなクロスフェード

### 🌍 多言語対応
- **日本語 / English / 中文（簡体字）**: 全コンテンツ対応
- **言語自動検出**: URLパラメータ → localStorage → ブラウザ言語設定 → デフォルト(日本語)
- **永続化**: ユーザー選択言語を localStorage に保存

### 📱 完全レスポンシブ
- デスクトップ、タブレット、スマートフォン対応
- タッチジェスチャー最適化
- モバイルファースト設計

### 🎯 主要セクション
- **Home**: HUDタイトルと回転ゲームカルーセル
- **Games**: モンスター図鑑風ゲームギャラリー（フィルタ・ソート機能付き）
- **Skills**: 技術スキルツリー表示
- **About**: プロフィール詳細
- **Timeline**: キャリアタイムライン
- **Updates**: 最新ニュース & 更新情報

### 🎮 ゲーム作品展示
- 15作品以上の制作ゲーム
- 制作時の学年表記
- App Store / Steam リンク対応
- 受賞作品表示（🏆バッジ）

### 🚀 技術機能
- **スムーススクロール**: easeOutCubic イージング関数による自然な加速・減速
- **Canvas 2D アニメーション**: 視差スクロール対応グリッド + 浮遊パーティクル
- **ゲーム詳細モーダル**: 豊富な画像・動画・説明表示
- **動的フィルタリング**: カテゴリ・技術スタックでの即座な絞り込み

## 技術スタック

### フロントエンド
- **TypeScript**: 型安全な開発
- **Vite**: 高速バンドル & HMR
- **CSS3**: Custom Properties、CSS Grid、Flexbox
- **Canvas 2D**: 背景アニメーション描画

### アーキテクチャ
- **Onion Architecture**: Domain → Application → Infrastructure → Presentation
  - ビジネスロジックと UI の分離
  - 外部ライブラリ依存性の最小化
  - テストのしやすさと保守性向上
- **Composition Root（main.ts）**: 依存性注入を一箇所で管理

### ツール
- **GitHub Pages**: 静的サイトホスティング
- **GitHub Actions**: 自動ビルド & デプロイ
- **ESLint + Prettier**: コード品質管理

## 📧 問い合わせ機能

EmailJSを使用した問い合わせフォーム：
- リアルタイムバリデーション
- 送信状態の視覚的フィードバック
- 自動返信メール対応
