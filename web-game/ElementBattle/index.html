Element Battle

ルール説明

Element Battleは自分の手札を使用し「Element」を強化して、先にElement Pointが20点に達したほうが勝ちです！

  • 1:各プレイヤーに５枚のカードを配ります
  • 2:山札から１枚カードを引き、手札から１枚カードを場に出します
  • 3:カードのマークに応じて「エレメント」が増減します
  • 4:Element Pointが20点に達したほうが勝ちです！

各マークの解説

自分のエレメントを
2ポイント強化します

他のプレイヤーのエレメントを
2ポイント減少させます

自分のエレメントを
1ポイント強化します

他のプレイヤーのエレメントを
1ポイント減少させます

特殊カード

Joker

以下の2つから選べます

自分のエレメントを5ポイント強化します

相手のエレメントを5ポイント減少させます

♠A

自分と相手のエレメントポイントが入れ替わります

♥7

自分のエレメントポイントを
3倍強化します

♦7

自分のエレメントポイントを
2倍強化します

♥K・♦K

自分のエレメントを
3ポイント強化します

♠K・♣K

相手のエレメントを
3ポイント減少させます

特殊ルール

Twin Suit

同じマークを2ターン連続で出したら
もう1枚カードが追加されます

※相手のターンで出したマークの次のターンで自分が同じマークを出しても効果は発動しません

Twin Rank

同じ数字を2ターン連続で出したら
もう2枚カードが追加できます

※相手のターンで出した数字の次のターンで自分が同じ数字を出しても効果は発動しません

※ルールは「H」キーでいつでも確認できます
